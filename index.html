Roll of the Dice

Welcome to yet another game of dice! You are probably wondering why was this game built? Well, the creator of this game was feeling kind of bored, so he decided to join a game jam hosted by GMTK. This year's theme is Roll of the Dice, therefore this game is all about dice. In the section below, you can find the existing 3 game modes, where we explain the objective of each mode, as well as its difficulty, waging constraints, and associated rewards.
