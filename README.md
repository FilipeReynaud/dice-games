# dice-games

The current game was developed within the context of the gmtk game jam (see: https://itch.io/jam/gmtk-jam-2022 ).

## Dependencies
- npm

## How to Run

To run this application, open a terminal and run the following

`npm start`

then, open a browser and go to

`localhost:1234`
